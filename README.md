# NekoVim
## Installation
Clone this repository in either %localappdata%\nvim (Win32) or ~/.config/nvim (Unix).
## Features
* Mason
* Lazy
* Fast as FUCK
* Tabs
* Sexy status line
* Notifications and separate command line so you don't strain your neck by looking down
* Start screen
* File explorer
* Code completion
* Code *snippets*
* Fuzzy finder
* Warmer version of Nord theme
* Keybinds
* Organisation
